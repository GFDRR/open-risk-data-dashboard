    Contextual Help Homepage

    The Open data for resilience index is an initiative aiming at Tracking and improving the state of the art of open data for resilience worldwide. The dashboard maps the existence of datasets considered key to conduct disaster risk assessment. The index scores the various countries in terms of data availability, accessibility and openness and represents a tool for advocacy for open data in risk analysis.
    For a complete description of the methodology behind the index consult the Final Report (Methodology).

    Contextual Help countries page

    This page presents the score of countries in map and tabular form. You can click on the Countries and you will be directed to a page that will enable you to browse the single datasets submitted for that Country. Scores are summarised per country or broken down for data categories. the scores can also be filtered using the hazard icons.
    For a complete description of the methodology behind the index consult the Final Report (Methodology).

    Contextual Dataset Details page

    This page enables the visualisation of each dataset instance details. if you are the owner of such dataset, or a user with administration capabilities, you can also access the edit page for this dataset. the edit page is identical to the submission page and allows you to change details regarding the dataset instance.
